### 留言板

无须自己搭建服务器，一个链接开启您的留言板

#### 特性

1. 支持markdown
2. 支持emoji
3. 支持代码高亮
4. 支持实时预览
5. 支持插入链接
6. 支持评论、回复
7. 支持生成唯一ID
8. 可以基于tc-comment进行二次开发

使用方式：

```
https://shiyix.cn/p/message-board?app=xxx
```

或

```
https://theajack.gitee.io/message-board?app=xxx
```

app 为您的应用id

github地址

